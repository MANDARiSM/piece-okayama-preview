# 相談支援事業所ピース ウェブサイトv2 — デザイン方向 確認用モック（社外秘）

クライアント確認用の静的モックです。トップページのデザイン方向 3 案を比較できます。

- `index.html` … 3案の比較トップ（ここから各案へ）
- `direction-a-watercolor.html` … 案A「やわらかな水彩」（情緒・共感／明朝）
- `direction-b-pieces.html` … 案B「つながるピース」（やさしさ・親近感／丸ゴシック）
- `direction-c-calm.html` … 案C「凛とした安心」（信頼・連携先向き／明朝）

※ 文章・写真はすべて仮（ダミー）です。検索除け（noindex / robots.txt）を設定しています。
